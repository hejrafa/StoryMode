# Story Mode

A World of Warcraft addon that turns the game's sprawling questlines into guided, story-driven experiences. Browse campaigns, track your progress chapter by chapter, and always know where to go next.

## What it does

WoW has incredible storylines buried under hundreds of quest markers. Story Mode surfaces them as curated campaigns you can browse, follow, and track.

- **World Map tab** — a Story Mode tab sits right next to Quests and Map Legend, matching the native UI. Click any storyline to track your next quest.
- **Settings panel** — browse questlines in a split-panel interface with chapter progress, next step guidance, and one-click tracking
- **Smart quest suggestions** — figures out which quest you should do next based on your actual progress, skipping gated or bypassed quests
- **Chapter progress** — two-column checklist with color-coded chapters (green = complete, yellow = active, gray = not started)
- **Active detection** — knows which storylines you're currently playing

## Usage

Open the settings panel:
```
/sm
```

Track the next quest in your active storyline:
```
/sm track
```

Or open the **World Map** and click the **Story Mode** tab on the side panel.

## Installation

1. Download or clone this repo
2. Place the `StoryMode` folder in:
   ```
   World of Warcraft/_retail_/Interface/AddOns/
   ```
3. Restart WoW or `/reload`
