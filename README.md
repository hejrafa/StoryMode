# Story Mode

A World of Warcraft addon that turns the game's sprawling questlines into guided, story-driven experiences. Browse campaigns, track your progress chapter by chapter, and always know where to go next.

## What it does

WoW has incredible storylines buried under hundreds of quest markers. Story Mode surfaces them as curated campaigns you can browse, follow, and track — right from the settings panel.

- **Browse questlines** in a clean split-panel interface
- **See chapter progress** at a glance with a two-column checklist
- **Track your next quest** with one click — sets a waypoint or supertrack pin automatically
- **Active detection** — knows which storylines you're currently playing

## Usage

Open the settings panel:
```
/sm
```

Track the next quest in your active storyline:
```
/sm track
```

Or open **Options → AddOns → StoryMode** from the game menu.

## Installation

1. Download or clone this repo
2. Place the `StoryMode` folder in:
   ```
   World of Warcraft/_retail_/Interface/AddOns/
   ```
3. Restart WoW or `/reload`
